
### OPERATORS 

|OPERATOR|DESCRIPTION|EXAMPLE|
|---|:---:|
| ""   | strict search / строгий поиск |
| *    | any text / любой текст |
| .    | any symbol / любой символ | 
| -    | exclude word / исключить слово | 
| ..   | range / диапазон |
| \|   | logical OR / логическое или |  
| ()   | group miltiple searches |
